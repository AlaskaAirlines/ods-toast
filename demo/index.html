This component has an available slot for placing custom content. If there are custom styling needs, please see this article for more information how to do so.

Requires @alaskaairux/ods-hyperlink: ^1.0.6

Additional utility styling is supported. The classes util_paddingLeft--none and util_paddingRight--none can be placed on the ods-hyperlink element using the class attribute.

This component is pre-coded to manage a mobile-first UI state.
